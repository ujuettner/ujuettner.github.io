## Install Jekyll

```bash
ruby --version   # has to be >= 2.1.0
gem install bundler
bundle install
```

## Update Gems (e.g Jekyll)

```bash
bundle update
```

## Local Build of Site

```bash
bundle exec jekyll serve
```

Open http://127.0.0.1:4000/ in browser.
